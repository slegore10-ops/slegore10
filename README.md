This program simulates a simple retail store POS system. It allows a cashier to log in and manage a shopping cart. The system is designed for a console based interaction and demonstrates basic inventory and transaction management.

Assumptions
. Only one cashier runs at a time.
. It will only run on the device it is installed on.
. Product catalog is static and predefined in the program.

Limitations
.Transactions are not saved for record keeping.
.Removing items from cart does not return them to stock.
.There is no storage so program resets once it ends.
.Cannot handle multiply customers simultaneously.
